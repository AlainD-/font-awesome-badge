Examples of Stacked Icons

fa-badge-bottom-plus on fa-folder
fa-badge-bottom-question on fa-database
fa-badge-bottom-question on fa-database
fa-badge-bottom-bolt on fa-database

To stack multiple icons, use the fa-stack and fa-badge-stack classes on the parent, the fa-stack-1x for the regularly sized icon, and also fa-stack-1x for the badge(s) icon. fa-inverse can be used as an alternative icon color. You can even throw larger icon classes on the parent to get further control of sizing.

A badge is usually composed of 2 icons: fa-badge-bottom-circle-surround stacked just after the main icon, in order to control the surround of the badge; and a second fa-badge-bottom-* wich represent the badge itself. Sometimes to give more control of the inner color of some badges it possible to stack inbetween a fa-badge-bottom-circle which size is a bit smaller than fa-badge-bottom-circle-surround. 

<span class="fa-stack fa-badge-stack fa-2x">
  <i class="fa fa-folder fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-circle-surround fa-inverse fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-plus fa-stack-1x text-success"></i>
</span>
fa-badge-bottom-plus on fa-folder<br>
<span class="fa-stack fa-badge-stack fa-2x">
  <i class="fa fa-database fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-circle-surround fa-inverse fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-question fa-stack-1x text-danger"></i>
</span>
fa-badge-bottom-question on fa-database<br>

Instead of the parent, the size of the stacked icon can be controled on each of the stacked icons themselves.

<span class="fa-stack fa-badge-stack">
  <i class="fa fa-database fa-stack-1x fa-lg"></i>
  <i class="fa-badge fa-badge-bottom-circle-surround fa-inverse fa-stack-1x fa-lg"></i>
  <i class="fa-badge fa-badge-bottom-question fa-stack-1x text-danger fa-lg"></i>
</span>
fa-badge-bottom-question on fa-database<br>

Herebelow a more complex example using a combination of 3 badges.

<span class="fa-stack fa-badge-stack fa-2x">
  <i class="fa fa-database fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-circle-surround text-danger fa-stack-1x"></i>
  <i class="fa-badge fa-badge-bottom-circle fa-stack-1x fa-inverse"></i>
  <i class="fa-badge fa-badge-bottom-bolt fa-stack-1x text-warning"></i>
</span>
fa-badge-bottom-bolt on fa-database<br>
It is important to notice that all the stacked icon must have the same size attributes in order to render the whole icon correctly.
fa-badge-stack is used to reduce the width and height of the stacked parent span. Where fa-stack creates a stack of 2em, fa-badge-stack creates a stack of 1em width and height.
